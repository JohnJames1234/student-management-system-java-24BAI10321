# 🎓 Student Management System (Java)

A console-based Java application to efficiently manage student records. This project demonstrates core Java concepts like Object-Oriented Programming (OOP), file handling, and user interaction through a menu-driven interface.

---

## 📌 Project Overview

The **Student Management System** allows users to perform CRUD operations (Create, Read, Update, Delete) on student data. All records are stored in a file to ensure data persistence.

---

## 🚀 Features

- ➕ Add new student  
- 📋 View all students  
- 🔍 Search student by ID  
- ✏️ Update student details  
- ❌ Delete student record  
- 💾 File storage (data persists after program exit)  

---

## 🛠️ Technologies Used

- Java  
- File Handling (BufferedReader, BufferedWriter)  
- Object-Oriented Programming (Classes & Objects)  
- CLI (Command Line Interface)

---

## ⚙️ How It Works

1. The program starts with a menu.  
2. User selects an operation (Add, View, Search, etc.).  
3. Data is processed using `StudentManager`.  
4. All student records are saved in `students.txt`.  
5. Data is automatically loaded when the program starts again.
